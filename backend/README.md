# Backend Architecture Note

The production backend source for HTP scoring, pattern detection, prompt construction, and premium report generation is intentionally redacted from this public portfolio repository.

This public copy keeps:

- Package metadata and environment example
- API and product documentation at the repository root
- Frontend integration points that show how the application consumes report APIs

The private backend includes:

- Authenticated Express APIs
- MySQL data access
- HTP feature extraction and pattern mapping
- Structured report-input generation
- LLM provider abstraction
- Premium report persistence and history retrieval
- Admin QA workflows

The redaction protects proprietary report-generation logic and avoids exposing sensitive psychological scoring rules while preserving enough architecture context for portfolio review.
